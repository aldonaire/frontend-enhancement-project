mission

Why we exist

Nexus was born from a simple frustration: the tools available to web developers were stuck in 2D. While gaming engines gave artists 3D worlds to build in, web developers were left with div soup and pixel-perfect layouts that felt lifeless by comparison.

We set out to change that. By combining the accessibility of HTML/CSS with the power of WebGL and Three.js, Nexus lets any developer — regardless of graphics programming experience — ship beautiful 3D interfaces that run in any browser.
