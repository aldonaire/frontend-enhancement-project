Workflow

Six steps to complete

Follow these steps carefully to successfully complete the project and submit your pull request.

STEP 01

Clone the repository

Download the project to your local machine using Git.

bash 
git clone <repo-url>
cd <repo-name>
STEP 02

Create your branch

Use the format: feature/<lastname>-frontend-enhancement

bash 
git checkout -b feature/donaire-ui-improvement
STEP 03

Make enhancements

Implement at least 3 improvements — UI/UX, functional, or optimization.

STEP 04

Commit changes

Use the feat: prefix for meaningful commits.

bash 
git add .
git commit -m "feat: improved navbar responsiveness"
STEP 05

Push your branch

Upload changes to the remote repository.

bash 
git push origin feature/<branch-name>
STEP 06

Create a pull request

Submit a PR to main with a summary, before/after screenshots, and features list.
