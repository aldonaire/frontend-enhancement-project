About the
Project

The Frontend Enhancement Project is a collaborative hands-on exercise focused on improving a starter web application using modern frontend practices and Git-based workflows.

Students clone the repository, create a personal feature branch, and implement meaningful improvements to the UI, UX, responsiveness, and code quality — then submit their work via Pull Request for review.

The goal is to experience the full development loop: from branching strategy and incremental commits to peer review and structured submission — mirroring real-world collaborative development.
