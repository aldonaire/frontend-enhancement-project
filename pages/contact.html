Before you write

Common answers

Can I push to main?

No. Always create a feature branch first using the naming convention provided.

How do I format commit messages?

Use feat: prefix followed by a short, specific description of the change.

What goes in a pull request?

Summary of changes, before/after screenshots, and a list of all features added.
